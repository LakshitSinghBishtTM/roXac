# Roadmap

## Status

roXac is feature-complete.

The project has reached its intended scope. No additional functionality is planned.

---

## Not Planned

The following are intentionally outside scope:

- Scientific functions
- Expression parsing
- Variables or memory
- Complex numbers
- Graphing
- Unit conversions
- GUI
- Network functionality
- Cloud synchronisation
- Plugin systems

---

## Maintenance Policy

Future updates will be limited to:

- Critical bugs
- Security vulnerabilities
- Dependency compatibility
- Packaging or installation issues

No feature releases are expected.
